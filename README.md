Консольное приложение "Библиотека"

Небольшое консольное приложение на C++ для управления личной библиотекой.  
Поддерживает работу с книгами и пользователями, сохранение данных в файл и интерактивное текстовое меню.

Возможности

- Просмотр всех книг.
- Просмотр всех пользователей.
- Добавление новой книги.
- Регистрация пользователя с уникальным ID и ограничением по числу книг.
- Выдача и возврат книг с проверкой доступности и лимитов.
- Поиск книги по ISBN.
- Просмотр профиля пользователя.
- Сохранение и загрузка данных из файла data/library_data.txt.
- Обработка ошибок и валидация ввода в консольном меню.

Сборка и запуск

Требования: компилятор с поддержкой C++17, CMake 3.10+.

mkdir build
cd build
cmake ..
cmake --build .

Запуск:

./library # Linux/macOS
.\library.exe # Windows

Использование

При запуске отображается меню:

1. Просмотреть все книги  
2. Просмотреть всех пользователей  
3. Добавить новую книгу  
4. Зарегистрировать пользователя  
5. Выдать книгу пользователю  
6. Принять книгу от пользователя  
7. Поиск книги по ISBN  
8. Просмотреть профиль пользователя  
9. Сохранить данные в файл  
10. Выход  

Приложение автоматически загружает данные из data/library_data.txt при старте (если файл существует) и сохраняет данные при выборе пункта 9.

Реализация

- Класс Book  
  Хранит название, автора, год, ISBN, признак доступности и ID пользователя, который взял книгу.  
  В конструкторе проверяет корректность года и непустой ISBN.

- Класс User  
  Хранит имя, ID пользователя, список взятых книг (ISBN) и максимальное допустимое количество книг.  
  Предоставляет методы для добавления/удаления книги и проверки, может ли пользователь взять ещё одну книгу.

- Класс Library  
  Управляет коллекциями книг и пользователей, реализует поиск книги по ISBN и пользователя по имени/ID, а также методы saveToFile и loadFromFile для работы с файлом.

Исключения (std::invalid_argument, std::runtime_error) используются для ошибок валидации и логических ошибок; они обрабатываются в main.cpp и выводятся понятные сообщения для пользователя.
